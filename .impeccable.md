# Impeccable — pointer

The canonical design system now lives in **[`DESIGN.md`](DESIGN.md)** (Stitch format:
machine-readable tokens + Overview / Colors / Typography / Elevation / Components /
Do's and Don'ts). Product/audience context lives in **[`PRODUCT.md`](PRODUCT.md)**.

Read those before any UI work. In short: Billboy is a Thai e-Tax/accounting app for
SMEs — **"The Trusted Companion (ผู้ช่วยคู่ใจ)"**. Navy authority + teal/gold warmth on a
light, softly-decorated surface. **The mascot and wallpaper are core brand identity —
never strip them for a "cleaner" minimal look** (a navy/white austere redesign was rejected).
Polish and elevate; keep the personality.
